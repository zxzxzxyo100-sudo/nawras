# 🦅 Al-Nawras CRM - النظام الكامل

## 📦 محتويات الملف

هذا الملف يحتوي على **نظام CRM متكامل** لشركة النورس للشحن.

### 📂 الهيكل:

```
nawras-crm-final/
├── 📄 index.html                    # صفحة البداية (افتحها أولاً)
├── 📚 README-FULL.md                # الدليل الكامل
├── 🔧 INSTALLATION.md               # دليل التثبيت خطوة بخطوة
├── 💾 database-schema.sql           # قاعدة البيانات
├── ⚙️ state-machine-engine.js       # محرك الحالات
├── 🌐 api/                          # Serverless Functions
│   ├── new-customers.js
│   ├── inactive-customers.js
│   └── orders-summary.js
├── ⚛️ react-crm/                    # React Dashboard
│   ├── src/
│   │   ├── components/              # مكونات React
│   │   ├── pages/                   # الصفحات
│   │   ├── contexts/                # Dark Mode
│   │   └── services/                # API Layer
│   ├── package.json
│   └── tailwind.config.js
├── 🔐 .env.example                  # مثال للمتغيرات
├── 📝 .gitignore
├── 🚀 netlify.toml                  # إعدادات Netlify
└── 📦 package.json
```

## 🚀 البدء السريع (3 خطوات)

### 1️⃣ افتح `index.html` في المتصفح
```bash
# في المجلد المفكوك
open index.html
# أو
firefox index.html
```

### 2️⃣ اقرأ دليل التثبيت
```bash
cat INSTALLATION.md
```

### 3️⃣ راجع الدليل الكامل
```bash
cat README-FULL.md
```
